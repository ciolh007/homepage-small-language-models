# Course Page: Small Language Models and Edge AI for Intelligent Robotics

This repository contains the public info for [Jacky Baltes'](https://scholar.google.com/citations?user=557QR3kAAAAJ&hl=en&oi=ao) course on
Small Language Models and Edge AI for Intelligent Robotics.

This course is offered in the Department of [Electrical Engineering](https://www.ee.ntnu.edu.tw/)
at the National Taiwan Normal University (NTNU) in Taipei, Taiwan.

Students can also participate via the Global Virtual Classroom program of NTNU.

You can view the homepage at [https://ciolh007.github.io/homepage-small-language-models](https://ciolh007.github.io/homepage-small-language-models)
